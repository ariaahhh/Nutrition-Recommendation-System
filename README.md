# Personalized Nutrition Recommendation System

A data-driven Python project that analyses Indian food datasets and delivers 
personalised dietary suggestions based on user health inputs.

## Features
- BMI calculator with health category classification
- Symptom-based nutrient deficiency predictor (Iron, Protein, Calcium, Vitamin C)
- Personalized food recommendations filtered by diet type (Vegetarian / Non-Vegetarian)
- Nutrient distribution visualizations using Matplotlib
- Data cleaning and preprocessing pipeline using Pandas

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Google Colab

## Project Structure
## How It Works
1. User enters age, gender, height, weight and diet type
2. System calculates BMI and classifies health category
3. User inputs symptoms or known deficiency
4. System predicts deficiency and filters food dataset
5. Top 5 recommended foods displayed with nutrient breakdown
6. Visual chart generated showing average nutrient values

- Data cleaning and preprocessing with Pandas
- Exploratory Data Analysis (EDA)
- Building logic-based recommendation systems
